# Quickstart (Frontend)

## Requisitos
- Node.js 18+

## Correr local
```bash
cd apps/frontend
npm install
npm run dev
```

Abre `http://localhost:3000`.

## Build
```bash
npm run build
npm run start
```

## Deploy (Vercel)
- Importa `apps/frontend` como proyecto Next.js.
- No requiere variables para el demo visual.
