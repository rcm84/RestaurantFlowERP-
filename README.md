# RestaurantFlowERP (v1)

Monorepo listo para vender como **suite full‑stack**:

- **5 landings premium** (restaurantes ficticios) pensadas como “showroom” para vender skins/plantillas.
- **Workflows n8n** (demos/automatizaciones) en `workflows/n8n`.
- **SQL base** para Supabase/Postgres en `db/`.
- **Extras** (miniaturas, videos, PDFs) en `extras/`.

## Estructura
- `apps/frontend` → Next.js (App Router) + Tailwind. Selector + 5 landings.
- `workflows/n8n` → JSON exportados de n8n.
- `db` → SQL.
